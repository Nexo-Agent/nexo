# Changelog 0.1.0-alpha.47

## Features & Improvements

### LLM Connections

- **Auto-test Connection**: Connection dialog now automatically tests the connection and fetches the model list while typing (with debounce).
- **Save Flow**: Performs a final connection check when saving to ensure models are up-to-date.
- **UI Improvements**: Refined the layout of connection cards for better visual consistency.

### Hub & Community

- **Unified Hub**: Migrated community components to a centralized Hub feature.
- **Consistency**: Implemented shared UI patterns and improved the responsive grid layout for community sections.
- **Loading States**: Added skeleton loading for a smoother experience while fetching community content.

### Agent Settings

- **Simplified Layout**: Reorganized the Agent settings screen to focus on the installation and management flow.

### Addon Management

- **Path Readability**: Long runtime paths are now displayed using middle ellipsis for better clarity.
- **Structural Refactoring**: Improved the internal structure of runtime configuration cards.

### UI/UX Refinements

- **Enhanced Update Dialog**: Release notes are now rendered as Markdown and the modal size has been increased for better readability.
- **Toast Notifications**: Refined styling and behavior of toast messages.
- **Control Polish**: Improved spacing and alignment in various control headers and dialog footers.
- **Chat Stability**: Prevented text selection leakage in message items.
