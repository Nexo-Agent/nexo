## Các tính năng mới

- **Đa ngôn ngữ (i18n)**: Hỗ trợ tiếng Việt cho trình chỉnh sửa Flow (Flow Editor).
- **Cài đặt lần đầu**: Cải thiện trải nghiệm người dùng, ưu tiên kết nối LLM.
- **Quản lý Environment**: Cải thiện quản lý runtime (Python/Node.js).
- **Giao diện**: Thêm chỉ báo active cho item trong sidebar.

## Sửa lỗi

- Sửa lỗi hiển thị thừa số 0 trong cài đặt Addon.
- Các cải tiến nhỏ về giao diện và hiệu năng khác.
