# Changelog v0.1.0-alpha.48

## Features

- **CI/CD**: Tích hợp GitHub Actions cho frontend, tự động chạy test và kiểm tra độ bao phủ (coverage) khi push code hoặc tạo Pull Request.

## Improvements

- **Testing**: Bổ sung bộ test case toàn diện cho các component UI quan trọng:
  - Workspace management (Selector, Settings, Add/Edit Dialogs).
  - Usage tracking (Charts, Logs, Overview).
  - Updater components.
- **Environment**: Đồng bộ hóa múi giờ UTC trong cấu hình Vitest, đảm bảo các snapshot test hoạt động ổn định trên cả Local và CI.

## Fixes

- Khắc phục lỗi lệch múi giờ trong các snapshot test của `UsageLogs`.
