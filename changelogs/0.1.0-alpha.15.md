# Changelog 0.1.0-alpha.15

## Features

- **UI/UX Enhancements**:
  - Implemented animated loading borders for Agent Cards.
  - Enhanced Usage Chart UI with gradients and better color palettes.
  - Centralized theme variables for consistent styling across light and dark modes.
  - Added support for custom provider icons (LiteLLM, Fireworks AI, etc.).
- **Developer Tools**:
  - Integrated CrabNebula DevTools for easier debugging.
  - Added support for filtering chat messages to show "thinking" and "tool call" messages in developer mode.

## Improvements

- **Refactoring**:
  - Continued migration to RTK Query for data fetching (LLM and MCP connections).
  - Improved Markdown rendering for tables and code blocks (synced with app theme).
- **Core**:
  - Refined agent response handling and metadata updates.
  - Optimized UV binary selection in build process.

## Fixes

- Fixed markdown table alignment issues.
- Fixed addon runtime loading state (node/python runtimes).
- Fixed agent tool execution routing.
